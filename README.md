# DSA Solutions

My solutions to data structures and algorithms problems, organized by platform then topic. Written in C++ (C++17, `g++ -std=c++17`).

## Structure

```
DSA/
├── AZ/                          # AlgoZenith problems
│   ├── Basic/
│   ├── Binary Search/
│   ├── Hashing/
│   ├── Maths/
│   ├── Prefix Sum/
│   ├── STL/
│   └── Two Pointers, Sliding Window/
├── LeetCode/                    # LeetCode problems
│   ├── Arrays/
│   ├── Backtracking/
│   ├── Binary Search/
│   ├── Dynamic Programming/
│   ├── Graphs/
│   ├── Greedy/
│   ├── Hashing/
│   ├── Heaps/
│   ├── Linked Lists/
│   ├── Math/
│   ├── Sliding Window/
│   ├── Stacks & Queues/
│   ├── Strings/
│   ├── Trees/
│   ├── Tries/
│   └── Two Pointers/
├── CF/                          # Codeforces 
└── CSES/                        # CSES Problem Set 
```

## Conventions

- **Language:** C++17
- **One problem per file.**
- **LeetCode files** will follow kebab-case (e.g. `two-sum.cpp`).
- **CF files** will use the problem ID as the filename (e.g. `1352-g.cpp`).
